# Note locale

Ce document est charge dans l'editeur quand on active un lien Markdown local.

## Objet

- verifier l'ouverture interne d'une page Markdown
- conserver le meme rendu que le document principal
- rester dans ElephantNote au lieu de naviguer brutalement hors de l'editeur

## Contenu

Vous pouvez revenir au document principal avec l'historique du navigateur ou reimporter le document de demonstration.
